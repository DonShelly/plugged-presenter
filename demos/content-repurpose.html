1) New Notion item

  • Status changes to "Ready to repurpose"
  • OpenClaw cron picks it up
Transform

2) Platform adaptation

  • Extract core message + CTA
  • Tailor tone per platform
Review

3) Approval gate

  • Drafts sent to founder in Telegram
  • Approve/edit per channel
Publish

4) Distribution + logging

  • Post to selected channels
  • Store output + metrics for reuse
